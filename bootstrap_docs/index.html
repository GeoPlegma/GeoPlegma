Less distortions

Discrete Global Grid Systems (DGGS) tesselate the surface of the earth into zones of equal area, minimizing spatial distortions.

Abstractions

GeoPlegma provides traits and APIs to implement DGGS in Rust.

Available DGGS

GeoPlegma currently supports the following DGGS: DGGAL, DGGR
